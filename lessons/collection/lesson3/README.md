Students should work in pairs and will each pair need the following equipment:

- Raspberry Pi
- Micro USB power adaptor

- USB keyboard
- USB mouse
- HDMI cable
- A monitor or TV
- Two **Male to Female** jumper wires (try [Pimoroni](http://shop.pimoroni.com/products/jumper-jerky))
